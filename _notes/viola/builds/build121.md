# Changes
## Browser
- Disabled auto-fading on smartphones
- Enabled edge-to-edge mode universally
- Removed usage of Find & View Source on special pages
- Added files address bar SSL lock state
- Fixed crash in custom tabs

## Settings
- Fixed color picker crash when rotating device orientation
- Introduced two experimental options:
    - External download manager
    - Multi-layered startpage background
- Fixed restarting application (for changing to Simple Toolbar)
- Fixed problem applying start page wallpaper
- Fixed illustrations for theme options

## Miscellaneous
- Fixed crash on ICS devices
- Size optimizations
