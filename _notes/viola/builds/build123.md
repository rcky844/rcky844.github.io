# Changes
## Browser
- Make shortcut creation unavailable to unsupported devices
- More downloaded file type support

## Favourites / History
- Fixed ordering of entries

## Settings
- Refactored update system
- Added reset option to ad servers dialog

## Bug fixes
- Fixed issues with edge-to-edge support
- Fixed shortcuts of same URLs
- Fixed opening downloaded files on some devices
- Fixed layout issue with fullscreen information dialog
- Fixed possible crash with custom user agents
