# Changes
All changes here are compared with Fernando 8.0.2, released 5 March 2026.

## Browser
- Introduced automatically fading floating toolbars for tablet UI
- Added simple toolbar (without expanding toolbar)
- Fixed issues with displaying Viola banner with start page wallpaper

## Settings
- New experience for seleting start page wallpapers
    - Support for solid colours
- Added placeholder options inside custom search address dialog
- Redesigned about information page

## Miscellaneous
- Show warning for JavaScript code from external applications (workaround for CVE-2023-42471)
- Fixed usage issues with address bar (emptying, suggestions disappearing)
- More comprehensive search suggestions error notifications
- Code optimizations and simplification
