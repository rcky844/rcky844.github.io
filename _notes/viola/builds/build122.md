# Changes
## Browser
- Improved ad blocker
    - Support merged hosts lists for better coverage
    - Improved efficiency of hosts storage + compression

## Favourites / History
- Major rework of Broha database (favourites / history)
    - Removed null values support
    - Refactored complexities in database access
- Browser history based search suggestions
- Searchable favourites and history

## Bug fixes
- Fixed opening some downloaded files
