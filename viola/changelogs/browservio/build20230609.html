Browservio Nightly 20230609

Released: 2023-06-09

Welcome to the Browservio Nightly, a variant of Browservio with the latest features in development released every so often. This variant will not replace the release variant and is intended for testing purposes, so expect bugs and crashes to occur at random. Be sure to report any bugs or crashes to the Issues tab. Happy Browservio-ing!

Changelogs

Changes:

  • Added non-functional placeholders for advanced tab management system
    • Added settings for switching between Recents-based tab management and placeholder
  • Added a work-in-progress start page (accessible via browservio://start)
  • Added support for customizing ad server list

Technical changes:

  • Initial support for loading with advanced tab management system
  • Switched clearing Preferences to custom preferences
  • Switched SwitchPreference to custom preferences
  • Improved ad blocking response to servers

Design changes:

  • Matched the colour of progress bars in both Browser and CustomTabs tabbies

Bug fixes:

  • #41 - NullPointerException in CustomTabs if url is null
  • #42 - Failure to open data URI
  • INTERNAL-04 Action bar is cut-off in reach mode
  • INTERNAL-05 Text going out of view bounds like "Browservio Debug" in about dialog
  • INTERNAL-06 Script execution is not detected as blocked properly
