Exotic 7.0 (build 54)

Released: 2024-08-03

Information

This build is a proof-of-concept for the upcoming Exotic 7.0 update. Many features still weren't added yet, or doesn't have full functionality, but getting this build out early can allow testers to provide feedback.

Changelogs

Features:

  • Introduced privileged pages, allowing for easier control of Viola internal URLs
  • Added icons to download files

Behavior changes:

  • Support ad blocker hosts using localhost
  • Added and started using getRealUrl() for finer filtering of URLs

Bug fixes:

  • Fixed searching
  • Fixed toolbar closing when features cannot be handled

Miscellaneous:

  • Refactor and cleaned up dialogs, popup menus, array adapters

Known bugs / vulnerabilities

  • Startpage / New tabs page is partially non-functional with JavaScript disabled.
  • An attcker may abuse the ViolaBrowser.focusInputBox() to cause DoS on the browser.
