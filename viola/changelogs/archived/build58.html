Exotic 7.0 (build 58)

Released: 2024-08-06

Information

This build is a proof-of-concept for the upcoming Exotic 7.0 update. Many features still weren't added yet, or doesn't have full functionality, but getting this build out early can allow testers to provide feedback.

Changelogs

Behavior changes:

  • Improved URI validation
  • Better error codes are now provided for buss:// websites
  • Updates are now downloaded to /data/data directory
  • Release dates for updates are now displayed
  • Ad servers list are now downloaded correctly
  • Added toast message to inform you when ad servers have downloaded

Bug fixes:

  • Fixed miscellaneous crashes in history and favorites
  • Fixed not being able to access chrome:// URIs directly
  • Fixed not being able to use Custom Tabs
  • Fixed in-app updating
  • Disabled access to view source on privileged URLs (e.g. Startpage)

Known bugs / vulnerabilities

Bussin WebX

  • Lua scripts do not work
  • Relative links do not work
