Exotic 7.0 (build 55)

Released: 2024-08-03

Information

This build is a proof-of-concept for the upcoming Exotic 7.0 update. Many features still weren't added yet, or doesn't have full functionality, but getting this build out early can allow testers to provide feedback.

Changelogs

Features:

  • Restored support for API 14 - 18
  • Added support for resolving Bussin WebX websites (buss://)

Behavior changes:

  • Startpage: Disabled non-functional search input box when JavaScript is disabled
  • Settings: Added support for multi-flavoured builds

Bug fixes:

  • Fixed view source
  • Fixed compatibility with other URI schemes
  • Fixed possible vulnerabilities with ViolaBrowser.focusInputBox().
    For more details on this, please refer to the last build's changelogs.

Notes for introducing new build flavors

This build reintroduced support for older devices running Android 4.0.x - 4.3.x, via a custom legacy flavor. This legacy build targets API 19, and has a minSdk of 14

Please note that the modern build flavor now has a minSdk of 21. Previous Android 4.4 KitKat users may need to update to the legacy build manually. This change is done in preparation for supporting more advanced features.

Known bugs / vulnerabilities

  • Features may crash the browser on legacy builds on older APIs.
  • Ad servers may fail to download.
